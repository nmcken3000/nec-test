# React/Next.js Developer Take-Home Test

## Task overview

Create a small application using Next.js and TypeScript. This is a take-home test to assess your approach to building a simple app that adds a new user. Try not to spend more than an hour or two on this.

This exercise aims to serve as a starting point, and we'll discuss your approach and how you could take it forward, so try to limit the time you spend on it. We're open to your ideas and questions when we meet.

## Specification

Create a page with a form to add new users.

If you're feeling adventurous, give these a try:

- User List Page: Display a list of users.
- User Profile Page: Show details of a selected user.

## API/Data

Use mock data.

## Form

Include the following fields with validation where appropriate.

- Full name (text, required).
- Age (required, must be 18+).
- Country (dropdown, required).
- Interests (checkboxes, at least one required).

## Technical requirements

- Include basic tests.
- Apply styling of your choice.
- Use any libraries you like to use.

## Submission instructions

- Upload to a public GitHub repository.
- Include a README with:
  - Setup instructions.
  - Reasons for any decisions (libraries, patterns).
  - Any other relevant information.
